You don't need to be a guru or a wizard to help Perl 6 development. Just some free time, and the willingness to be part of a great community. There are a myriad of tasks for every skill set.

Most of them require coordination and help from the rest of the community, so get in touch with us.

Below are some ideas what you could do to help us. The list is by no means complete.

Programming Tasks

  • Write a module and add it to the proto module list.
  • Write tests
  • Help writing a compiler
  • Find, report and fix bugs
  • Help building a module infrastructure

Other Tasks

  • Write documentation
  • Blog about Perl 6
  • Write mailing list summaries
  • Organize Perl 6 conferences or hackathons
  • Raise funds for Perl 6 development
  • Answer questions on IRC, mailing lists and perlmonks
  • Help to administer services (IRC bots, source code repositories, servers)
